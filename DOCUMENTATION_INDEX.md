## 📞 How to Update Information

### **Update Company Contact Details:**
Edit: `/frontend/src/components/CompanyHeader.jsx`
```jsx
const companyInfo = {
  name: "Your Company Name",
  address: "Your Address",
  phone: "Your Phone",
  email: "Your Email",
  gstin: "Your GSTIN",
  categories: "Your Categories",
  tagline: "Your Tagline"
};
```

### **Update Menu Items:**
Edit: `/frontend/src/data/menu.json`
```json
{
  "slNo": 1,
  "itemName": "Item Name",
  "category": "Category Name",
  "veg_nonveg": "Veg/Non-Veg",
  "prices": [
    { "unit": "kg", "units": 1, "price": 500 }
  ]
}
```

### **Update Colors:**
Edit: `/frontend/src/index.css` (CSS variables section)
```css
:root {
  --primary-color: #f5ba4a;
  --primary-light: #ffc757;
  /* Update other colors here */
}
```
