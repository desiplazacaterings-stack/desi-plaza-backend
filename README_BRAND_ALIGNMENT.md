## 💾 How to Update

### **Update Business Info:**
```javascript
// Edit: /frontend/src/components/CompanyHeader.jsx
const companyInfo = {
  name: "Desi Plaza Caterings",
  address: "Your Address Here",
  phone: "Your Phone Here",
  email: "Your Email Here",
  gstin: "Your GSTIN Here",
  categories: "Weddings • Events • Corporate Catering",
  tagline: "Premium Catering Services for Your Special Events"
};
```

### **Update Menu:**
```json
// Edit: /frontend/src/data/menu.json
[
  {
    "slNo": 1,
    "itemName": "Item Name",
    "category": "Category Name",
    "veg_nonveg": "Veg/Non-Veg",
    "prices": [
      { "unit": "kg", "units": 1, "price": 500 }
    ]
  }
]
```

### **Change Colors:**
```css
/* Edit: /frontend/src/index.css */
:root {
  --primary-color: #f5ba4a;
  --primary-light: #ffc757;
  /* Update colors here */
}
```
