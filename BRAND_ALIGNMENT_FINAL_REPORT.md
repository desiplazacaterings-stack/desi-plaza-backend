## 💡 Quick Reference

### **How to Update Business Info:**
Edit `CompanyHeader.jsx` and update the `companyInfo` object:
```jsx
const companyInfo = {
  name: "Your Company",
  address: "Your Address",
  phone: "Your Phone",
  email: "Your Email",
  gstin: "Your GSTIN",
  categories: "Your Categories",
  tagline: "Your Tagline"
};
```

### **How to Add More Menu Items:**
Edit `menu.json` and follow the existing format:
```json
{
  "slNo": 25,
  "itemName": "Item Name",
  "category": "Category",
  "veg_nonveg": "Veg/Non-Veg",
  "prices": [
    { "unit": "kg", "units": 1, "price": 500 }
  ]
}
```

### **How to Change Colors:**
Update CSS variables in `index.css`:
```css
:root {
  --primary-color: #f5ba4a;
  --primary-light: #ffc757;
}
```
